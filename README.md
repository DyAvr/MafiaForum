# MafiaForum
## Описание проекта
    Проект представляет собой форум по спортивной мафии для новичков и любителей этой интеллектуальной игры. Форум будет предоставлять ответы на большинство вопросов для улучшения качества игры пользователей.  
    Форум содержит в себе такие элементы: тема, запись, ответ. На форуме присутствуют личные профили пользователей и рейтинг пользователей.
## Инструкция по применению
    1. Для установки нужно скачать zip архив с github репозитория.  
    2. Затем запустить MafiaForum.sln.  
    3. После заменить строку подключения к бд ("DefaultConnection") в файлах appsettings.json и appsettings.Development.json.
